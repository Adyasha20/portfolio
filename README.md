# Adyasha Priyadarshini Personal Portfolio

A personal portfolio website showcasing my skills, projects, certifications, and interests in Cybersecurity, Software Development.

## 🚀 Live Demo

🔗 Portfolio Website: https://adyasha20.github.io/portfolio/

## Overview

This is a fully responsive personal portfolio website built using HTML, CSS, and JavaScript. It showcases my education, technical skills, projects, certifications, and interests in Cybersecurity, Software Development. The website serves as a professional platform for recruiters, collaborators, and fellow learners to explore my work and connect with me.

## 🔗 Key Features

* **Modern & Responsive Design:** Optimized for desktops, tablets, and mobile devices for a seamless viewing experience.

* **Cybersecurity-Themed Interface:** Features a visually engaging design inspired by cybersecurity concepts and digital systems.

* **Interactive Project Showcase:** Includes detailed project information with an interactive modal displaying project descriptions, implementation details, and demonstration videos.

* **Certification Verification:** Provides direct access to certification PDFs for easy verification and credibility.

* **Technical Skills Dashboard:** Highlights programming languages, cybersecurity concepts, networking skills, and development tools.

* **Professional Networking:** Includes direct links to GitHub, LinkedIn, and email for easy communication and collaboration.

* **GitHub Pages Deployment:** Hosted using GitHub Pages for fast, reliable, and accessible online viewing.

## 💻 Tech Stack
Frontend: HTML5, CSS3 (Bento Grid Layout), JavaScript

Security Focus: Data Encryption, Secure File Transfer, Networking

Tools: Java Swing, HandBrake (Video Optimization), Git/GitHub
